DFXP 1.0 with caffeine

By $Author: plehegar $, $Date: 2009/11/01 18:08:36 $

This demonstration shows several features of DFXP (which is still work in progress by the way):

  • Regions: default region, overlays, positioning
  • Text effects: color, italic, bold, size, fonts
  • Timing effects: applied on regions, paragraphs, and words.

For feedback, please write to public-tt@w3.org (publicly archived). The DFXP source code is of course available. A screencast of the video is available on Youtube (without sound).

Note: this demonstration uses the HTML 5 video element, its media API, and a totally unconventional <text> element.
